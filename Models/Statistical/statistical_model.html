Abstract Statistical Model¶

The StatisticalModel class is an abstract base class used for all statistical models used for time series prediction. It represents the interface used by these models.

class MemoryAutoScaling.Models.Statistical.StatisticalModel(train_prop=0.6, val_prop=0.2)[source]¶

A statistical model for a data trace.

Parameters
train_prop: float

A float in the range [0, 1] representing the proportion of data in the training set. The default value is 0.6.

val_prop: float

A float in the range [0, 1] representing the proportion of data in the validation set. The default value is 0.2.

Attributes
_train_prop: float

The proportion of data in the training set.

_val_prop: float

The proportion of data in the validation set.

_model: Object

The underlying model being fit to the trace data.
