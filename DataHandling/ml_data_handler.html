ML Data Handler¶

The MLDataHandler is a class used to pre-process data so that the data can be used for a machine learning model. It is specifically built for handling time series data.

class MemoryAutoScaling.DataHandling.MLDataHandler(feature_vars, target_vars, train_prop=0.6, val_prop=0.2)[source]¶

Used to pre-process data to be used in a machine learning model.

The handler is designed for time series data and splits the data so that time series models are trained on the first train_prop proportion of the data and tested on the remaining 1 - train_prop proportion.

Parameters
train_prop: float

A float in [0, 1] representing the proportion of data used in the training set. The default value is 0.6.

feature_vars: list

A list of strings representing the names of features.

target_vars: list

A list of strings representing the names of the target variables.

Attributes
_train_prop: float

The proportion of data to be used in the training set.

_feature_vars: list

The names of the feature variables.

_target_vars: list

The names of the target variables.
