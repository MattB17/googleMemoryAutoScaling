Google Trace¶

The Trace class stores the data associated with a Google trace from the Borg cluster.

class MemoryAutoScaling.DataHandling.Trace(trace_id, start_time, end_time, ts_df, trace_usage, agg_window)[source]¶

Serves as a container for the data relating to a trace.

Parameters
trace_id: int

The ID for the trace.

start_time: int

The time at which the trace data begins

end_time: int

The time at which the trace data ends

ts_df: pd.DataFrame

A pandas DataFrame containing the time series data for the trace.

trace_usage: TraceUsage

A TraceUsage object representing usage information for the trace.

agg_window: int

An integer representing the length of the aggregation window. That is, for every agg_window periods the trace data is aggregated.

get_resource_utilization(target_col)[source]¶

The utilization of target_col for the trace.

The resource utilization for a time point is the average usage of target_col for that time point divided by the resource units allocated in that period. The total resource utilization is the average resource utilization over all time points.

Parameters
target_col: str

A string identifying the resource for which the utilization is calculated, either memory or CPU.

Returns
float

A float representing the utilization rate of resource_col for the trace.

get_spare_resource_in_window(target_col, win_start, win_end)[source]¶

The spare amount of target_col in [win_start, win_end].

The spare amount of target_col at a particular time point is the amount available minus the amount used at that time point. The total spare for the window is calculated as the sum of these values over all time points in the window, excluding win_end.

Parameters
target_col: str

A string identifying the resource of interest, either memory or CPU units.

win_start: int

An integer representing the start index of the window.

win_end: int

An integer representing the end index of the window.

Returns
float

A float representing the total spare amount of target_col for the window.
